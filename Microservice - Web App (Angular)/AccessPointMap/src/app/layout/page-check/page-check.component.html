Security types

  • WPA2 - The strongest security standard, WPA2 type uses CCMP / AES encryption.
  • WPA - A slightly older type of security, using TKIP / RC4 and Michael (MIC) encryption. It is recommended to switch to WPA2.
  • WPS - The WPS system automatically generates the parameters needed for connection. Misconfiguration can be a serious vulnerability.
  • WEP - This is a type of security developed in 1999. WEP is vulnerable to FMS attack which makes it necessary to switch to WPA / WPA2 security.
  • None - The network is not secured. Anyone can steal valuable data. WPA / WPA2 security must be configured.

Nowadays, it is imperative to ensure network security. Ubiquitous digitization means that communication, payments, etc. are carried out via the Internet. Poor-quality security allows an unauthorized person to view the traffic in our home network. Logins, passwords, bank login details etc. are at risk.
